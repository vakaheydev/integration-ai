# 🚀 Быстрый старт

## Минимальные шаги для запуска

### 1️⃣ Установите зависимости
```bash
npm install
```

### 2️⃣ Запустите приложение
```bash
npm run dev
```

### 3️⃣ Откройте в браузере
```
http://localhost:3000
```

---

## 🎯 Тестирование без backend

Если backend еще не готов, вы можете:

1. **Изучить интерфейс** - все компоненты отображаются корректно
2. **Проверить валидацию** - попробуйте загрузить не-JSON файл
3. **Посмотреть код** - все исходники в папке `src/`

---

## 🔧 Настройка backend URL

Отредактируйте файл `.env`:

```env
VITE_API_BASE_URL=http://localhost:8080/api
```

Замените на URL вашего backend сервера.

## ❓ Возникли проблемы?

### Ошибка при npm install
```bash
# Очистите кеш и попробуйте снова
npm cache clean --force
Remove-Item -Recurse -Force node_modules
npm install
```

### Приложение не открывается
1. Проверьте, что порт 3000 свободен
2. Посмотрите вывод в терминале на наличие ошибок
3. Попробуйте очистить кеш браузера

### Ошибки TypeScript
```bash
# Проверьте компиляцию
npx tsc --noEmit
```
